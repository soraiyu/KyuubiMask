# Fastlaneスクリーンショット

このディレクトリにはアプリストアとF-Droid用のスクリーンショットが含まれます。

## 必要な構造
- `phoneScreenshots/`: スクリーンショットをこのディレクトリに配置します。
- `featureGraphic.png`: フィーチャーグラフィックファイルをここに配置します。

スクリーンショットは1080x1920、フィーチャーグラフィックは1024x500を推奨します。

## ガイドライン
- PNGまたはJPEG形式を使用
- アプリの主要機能を表示
- 画像は明瞭でプロフェッショナルに
- 個人情報や機密情報を含めない
